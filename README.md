# YTSpeedHotkey

This [Tampermonkey](https://tampermonkey.net/) script adds the ability to use the hotkey `Q` to change the playback speed on [Youtube](https://www.youtube.com/).

## Installation

First [install Tampermonkey](https://tampermonkey.net/) and then click [here](https://github.com/DerEnderKeks/YTSpeedHotkey/raw/master/ytspeedhotkey.user.js).

## Usage

Press `Q` while on Youtube to toggle the playback speed between 1 and 2.
